# Getting Started with Building this application

## Required Software
 - Git
 - node

## Steps
1. Cloning this repo
```
git clone https://github.com/sahilarora3117/MdTex
```
2. Move into the folder 
```
cd MdTex
```
3. install the dependencies
```
npm install
```
4. start the application
```
npm start
```
